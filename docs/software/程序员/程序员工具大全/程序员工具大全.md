1、nodejs版本管理器，版本切换

Node Version Manager（NVM）是一个常用的 Node.js 版本管理工具，可用于在开发过程中方便地切换不同版本的 Node.js。通过 NVM，用户可以根据项目需求选择不同的 Node.js 版本，而无需手动安装和卸载多个版本的 Node.js。
