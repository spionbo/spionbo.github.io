## 准备工作

开始之前，需要准备以下工具才能下载网盘内容

1. chrome浏览器（其它浏览器也可以，我这里是以chrome浏览器为教程讲解）
2. 油猴（tampermonkey）浏览器扩展程序
3. 网盘下载插件
4. IDM下载器

## 安装
1、安装chrome浏览器；
2、安装油猴（tampermonkey）浏览器扩展程序
3、安装网盘下载插件-网盘直链下载助手
4、安装IDM下载器
